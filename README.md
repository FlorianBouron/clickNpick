## How do I run this?
* Run `npm install` from this folder.
* Run `npm run build.` from this folder.
* Run `npm run start` from this folder or `yarn start` if you are using `yarn`.
* Open [http://localhost:1337/admin/](http://localhost:1337/admin/)
